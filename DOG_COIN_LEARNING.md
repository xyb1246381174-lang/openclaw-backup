## 💡 什么是"土狗"？

### 土狗 = "Rug Pull" 的音译

```
土狗 = Rug Pull
中文 = 拉地毯 / 跑路
含义 = 项目方卷款跑路
```

---

## 🎯 土狗币是什么？

### 定义
```
土狗币 = 诈骗性质的山寨币
特点 = 高风险、高回报承诺
风险 = 99% 会归零
```

### 典型特征

| 特征 | 说明 |
|------|------|
| 新币种 | 刚上线几天/几周 |
| 高APY | 承诺1000%+回报 |
| 匿名团队 | 不知道创始人是谁 |
| 低市值 | 几千到几万美元 |
| Telegram营销 | 疯狂宣传 |

---

## 🚨 土狗币的常见套路

### 1. 貔貅币（Piggy）

```
特点：
- 只能买，不能卖
- 项目方 rug pull
- 散户被套牢

信号：
- 流动性被锁定
- 只有买入没有卖出
```

### 2. 貔貅盘

```
操作：
- 项目方操控价格
- 只涨不跌（假的）
- 最后直接归零

散户看到的是：
- 账面盈利
- 无法提现
```

### 3. 蜜獾币（Honey Pot）

```
特点：
- 任何人可以买
- 但不能卖（只有项目方能卖）

原理：
- 智能合约有后门
- 只有项目方能转移资金
```

## 🎭 土狗项目常见套路

### Step 1: 造势

```
- Telegram群拉人
- Twitter疯狂宣传
- KOL喊单
- 制造FOMO情绪
```

### Step 2: ico/sale

```
- 预售代币
- 承诺高回报
- 限时优惠
- 紧迫感
```

### Step 3: 上线

```
- 小幅上涨
- 让散户盈利
- 继续宣传"会涨"
```

### Step 4: Rug Pull

```
- 项目方抛售
- 价格暴跌
- 散户被套
- Telegram群解散
- 网站关闭
```

## 🔍 如何识别土狗币？

### 危险信号 🚩

| 信号 | 风险等级 |
|------|---------|
| 匿名团队 | 🟠 高 |
| 无审计 | 🟠 高 |
| APY>1000% | 🔴 极高 |
| 新币种 | 🟡 中 |
| Telegram群限流 | 🟠 高 |
| 只能买不能卖 | 🔴 极高 |
| 流动性未锁定 | 🟠 高 |

### 安全检查 ✅

```
1. 查看审计报告
2. 验证团队身份
3. 检查流动性是否锁定
4. 看合约是否开源
5. 检查持币地址分布
6. 搜索项目方历史
```

## 💰 土狗币 vs 正规币

| 对比 | 土狗币 | 正规币 |
|------|--------|--------|
| 团队 | 匿名 | 公开 |
| 审计 | 无 | 有 |
| 流动性 | 可操控 | 锁定 |
| 目的 | 割韭菜 | 建设 |
| 风险 | 归零 | 波动 |
| 法律 | 违法 | 合法 |

## 🛡️ 如何保护自己？

### 投资原则

```
1. 只投资你能承受亏损的钱
2. DYOR（自己研究）
3. 不要FOMO
4. 分散投资
5. 设置止损
```

### 避坑口诀

```
看到土狗快跑路
高APY是陷阱
团队匿名不要碰
流动性锁定才安全
```

### 黄金法则

```
✅ 只投正规交易所的币
✅ 只投有审计的项目
✅ 只投团队公开的币
✅ 只投流动性锁定的币

❌ 不投Telegram群的币
❌ 不投APY>100%的币
❌ 不投只能买不能卖的币
❌ 不投新上线的币
```

## 📈 如果真的想玩土狗

### 低风险玩法

```
1. 只用亏得起的钱
2. 小仓位（总资金1-5%）
3. 快进快出（上线就卖）
4. 设置止损线（-50%必走）
5. 不追高
```

### 记住

```
土狗币不是投资
是赌博
十赌九输
```
